## 🧪 Projeto em destaque

### 📊 ECG com ESP32 + AD8232

Sistema embarcado para aquisição e processamento de sinais cardíacos em tempo real.

🔧 Características técnicas:

* Frequência de amostragem: 250 Hz
* Leitura via ADC (12 bits)
* Transmissão de dados via UART
* Processamento de sinais no MATLAB
* Aplicação de filtros digitais (FIR e IIR)

🎯 Objetivo:
Desenvolver uma ferramenta para auxiliar no ensino de interpretação de ECG.
